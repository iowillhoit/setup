# Setup

## INIT
- Open `terminal`
- Type `git`
  - Will be prompted to install Dev Tools
- Install `brew`
  - Run: `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`

- Sync settings (~/Dropbox/applications/alfred)
